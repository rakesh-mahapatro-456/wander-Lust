## `$ cat overview.txt`

Wanderlust is a full-stack MVC travel listings app. Built around clean server-rendered patterns with geospatial data storage via LocationIQ geocoding, a Cloudinary image upload pipeline, Passport-Local-Mongoose session auth, and referential integrity through cascading deletes.

```
  POST /listings  ‚Üí  Joi validation ‚Üí Cloudinary upload ‚Üí LocationIQ geocode
                                              ‚Ēā
                              ‚ĒĆ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚Ēī‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚Ēź
                        image { url, filename }          GeoJSON coords
                        stored in MongoDB                stored in MongoDB
                              ‚Ēā                                ‚Ēā
                        served via CDN               MapLibre + MapTiler
                                                      renders map on page
```

## `$ cat features.txt`

### ūüĒź Authentication (Passport-Local-Mongoose)
```
POST /signup    ‚Üí  create user + hash password
POST /login     ‚Üí  validate + create session (secure cookie)
GET  /logout    ‚Üí  destroy session

Session middleware on every protected route
Unauthorized   ‚Üí redirect to /login
```

### ūüŹě Listings (Full CRUD)
```
GET    /listings       ‚Üí  all listings
GET    /listings/:id   ‚Üí  single listing + map
POST   /listings       ‚Üí  create (auth required + Joi validation)
PUT    /listings/:id   ‚Üí  update (owner only)
DELETE /listings/:id   ‚Üí  delete + cascade reviews (owner only)
```

### ūüďć Geospatial Data
```
Create listing ‚Üí LocationIQ geocoding API ‚Üí GeoJSON coordinates
                                                  ‚Ēā
                                       stored in MongoDB
                                                  ‚Ēā
                              rendered as interactive map
                              via MapLibre GL + MapTiler tiles
```

### ūüĖľ Cloudinary Image Pipeline
```
Upload form ‚Üí multer-storage-cloudinary (single image)
                          ‚Ēā
              { url, filename } ‚Üí MongoDB
                          ‚Ēā
              served via Cloudinary CDN
```

### ‚úćÔłŹ Review System
```
POST   /listings/:id/reviews            ‚Üí  add review (auth required)
DELETE /listings/:id/reviews/:reviewId  ‚Üí  delete (author only)

Cascade: listing deleted ‚Üí all linked reviews auto-removed
         (referential integrity via Mongoose middleware)
```

### ūüďä Data Models
```
Listing
  ‚Ēú‚ĒÄ‚ĒÄ title, description, price
  ‚Ēú‚ĒÄ‚ĒÄ location, country
  ‚Ēú‚ĒÄ‚ĒÄ image: { url, filename }   ‚Üź Cloudinary (single image)
  ‚Ēú‚ĒÄ‚ĒÄ geometry: GeoJSON          ‚Üź LocationIQ coords
  ‚Ēú‚ĒÄ‚ĒÄ owner: ref ‚Üí User
  ‚ĒĒ‚ĒÄ‚ĒÄ reviews: [ref ‚Üí Review]

Review
  ‚Ēú‚ĒÄ‚ĒÄ rating, comment
  ‚ĒĒ‚ĒÄ‚ĒÄ author: ref ‚Üí User

User
  ‚Ēú‚ĒÄ‚ĒÄ username, email
  ‚ĒĒ‚ĒÄ‚ĒÄ hash + salt (Passport-Local-Mongoose)
```

## `$ cat setup.txt`

```bash
git clone https://github.com/rakesh-mahapatro-456/wander-Lust.git
cd wander-Lust
npm install
```

Create `.env`:
```env
ATLASDB_URL=your_mongodb_atlas_url
SECRET=your_session_secret
CLOUD_NAME=your_cloudinary_cloud_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret
LOCATIONIQ_API_KEY=your_locationiq_key
MAPTILER_API_KEY=your_maptiler_key
```

```bash
node app.js
# App running at http://localhost:8080
# (or add "start": "node app.js" to package.json scripts)
```
